Experiment 9(a): NumPy and Pandas

AIM:
To write a Python program to implement NumPy and Pandas packages.

ALGORITHM:
1. Import NumPy and Pandas libraries.  
2. Create a NumPy array.  
3. Perform basic operations like mean and sum.  
4. Create a dictionary with sample data.  
5. Convert the dictionary into a Pandas DataFrame.  
6. Display the DataFrame.  
7. Calculate average of a column.
