AIM:
To implement the use of Regular Expressions (re module) in Python for pattern matching, searching, finding, substitution, splitting, and validation.

ALGORITHM:
Step 1:  Import the re module.
Step 2:  Define a text string containing a phone number and email.
Step 3:  Use re.match() to check if the string starts with a given pattern.
Step 4:  Use re.search() to find a phone number in the string.
Step 5:  Use re.findall() to extract all digits from the string.
Step 6:  Use re.finditer() to find digits along with their positions.
Step 7:  Use re.sub() to replace digits with *.
Step 8:  Use re.split() to split the string based on spaces.
Step 9:  Take date input from the user.
Step 10: Define a pattern for date format (dd/mm/yyyy).
Step 11: Use re.fullmatch() to validate the date format.
Step 12: Display whether the date is valid or invalid.
